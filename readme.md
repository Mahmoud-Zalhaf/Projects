# 🛒 Supermarket Sales Analytics Dashboard

## 📊 Project Overview

This project analyzes supermarket sales performance using SQL and Power BI.

The objective was to transform raw transactional data into actionable business insights through data cleaning, modeling, DAX calculations, and dashboard development.

---

## 🛠 Tools & Technologies

- SQL
- Power BI
- DAX
- Data Modeling
- Figma

---

## 🔄 Project Workflow

### 1. Data Preparation
- Cleaned and transformed raw data using SQL
- Handled missing values and inconsistencies
- Created reusable SQL Views

### 2. Data Modeling
- Built Fact and Dimension tables
- Implemented Snowflake Schema
- Created relationships between tables

### 3. DAX Measures
- Total Sales
- Profit
- Profit Margin
- Return Rate
- MoM Growth
- YoY Growth
- Sales Target Tracking

### 4. Dashboard Development
- Executive Overview
- Product Analysis
- Customer & Store Analysis
- Insights & Recommendations

---

## 📈 Key Insights

- Sales exceeded annual target by 5.8%
- North West region generated the highest sales
- Q4 contributed the highest revenue
- Return rate remained below 3%
